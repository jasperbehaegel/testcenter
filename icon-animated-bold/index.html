Icon Animated Bold

Copy of icon-animated-v2 but with twice as tick elements.

Icon set build with css only. Icons can be animated into a different state.

Consist of only a <div> (no inner <div> required).

Dimensions need to be set onto a container element. Set color as background-color on icon element self.

  • toggle icon size (large)
  • toggle icon type (menu)
  • toggle icon colour (grey)
