Hero Banner

Image banner, stretching to the full width of the viewport and visible in a nice way in all viewport sizes (mobile -> desktop)

See how the 50% doesn't completely fill the container box in table or mobile view. This is because the aspect-ratio of this particular image is less then 2:1

Top example: with background image: flexibility in positioning, but not in filesize.

Bottom example: with <img> tag and srcset attribute: flexibility in image file size and centered with a transform.

Explanation of the srcset attribute: http://martinwolf.org/2014/05/07/the-new-srcset-and-sizes-explained/

Browser support: good, see http://caniuse.com/#search=srcset
